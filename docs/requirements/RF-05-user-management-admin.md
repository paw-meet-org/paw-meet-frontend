# RF-05 - User Management (Admin)

## Goal
Allow administrators to manage platform users lifecycle.

## Scope
- Create users.
- Update users.
- Read/list users.
- Delete users.

## Acceptance Criteria
- Admin can create a user with valid required fields.
- Admin can edit user data and save changes.
- Admin can retrieve user details and list users.
- Admin can delete users with appropriate confirmation.
- Non-admin roles cannot access admin user-management actions.
