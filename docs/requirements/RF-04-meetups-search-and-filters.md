# RF-04 - Meetups Search and Filters

## Goal
Help users discover relevant meetups based on place, time, and pet compatibility.

## Scope
- Filter meetups by location.
- Filter meetups by date.
- Filter meetups by pet type.
- Sort results by proximity, popularity, or date.
- Invite users to meetups when capacity allows.

## Acceptance Criteria
- Users can apply each filter independently.
- Users can combine multiple filters and get coherent results.
- Sorting can be changed without losing active filters.
- Invite action is blocked when meetup capacity is full.
- Empty results are handled with clear user feedback.
