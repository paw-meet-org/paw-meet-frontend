# RF-09 - Sponsors Management

## Goal
Allow administration of sponsors and sponsored events in the platform.

## Scope
- Create sponsor.
- Update sponsor data.
- Retrieve sponsor information.
- Delete sponsor.

## Acceptance Criteria
- Admin can create a sponsor with required business fields.
- Admin can edit sponsor information and persist changes.
- Sponsor list/detail retrieval works for authorized roles.
- Admin can remove sponsors with confirmation and proper validation.
