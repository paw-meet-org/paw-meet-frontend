# RF-07 - Content Moderation

## Goal
Provide moderation tools to keep forum content aligned with community rules.

## Scope
- Remove posts that violate moderation policy.
- Temporarily ban users who repeatedly violate rules.

## Acceptance Criteria
- Authorized moderators/admins can remove violating content.
- Moderation actions are blocked for non-authorized users.
- Banned users lose posting privileges for the configured period.
- Moderation actions are traceable through system records/logs.
