# RF-01 - Profile and Pets

## Goal
Allow users to create and maintain personal profile data and linked pet information.

## Scope
- Create profile with basic data (name/username, email, password, bio).
- Edit profile information.
- Upload or change profile picture.
- Configure alerts/notifications for relevant events.
- Register and edit pet information (type, name, photo, description).

## Acceptance Criteria
- A new user can complete profile creation successfully.
- A logged-in user can edit profile fields and persist changes.
- Profile picture upload/change is supported with validated file formats.
- Notification preferences can be enabled or disabled.
- A user can create and edit one or more pets linked to their account.
